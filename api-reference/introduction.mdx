This reference documents all available endpoints in the Harmonic REST API. For GraphQL, see the [GraphQL overview](/graphql/overview).

## Base URL

```
https://api.harmonic.ai/
```

## Authentication

Pass your API key via the `apikey` header or query parameter. See [Authentication](/guides/authentication) for details.

## Endpoints

| Section | Description |
|---------|-------------|
| [Enrich](/api-reference/enrich) | Enrich companies and people by URL or identifier |
| [Search](/api-reference/search) | Search for companies, people, and investors |
| [Fetch](/api-reference/fetch) | Retrieve detailed company and person profiles |
| [Financing](/api-reference/financing) | Access investor and deal data (GraphQL only, paid add-on) |
| [Network](/api-reference/network) | Discover team connections to companies and people |
| [Company Lists](/api-reference/company-lists) | Manage company watchlists and custom fields |
| [People Lists](/api-reference/people-lists) | Manage people watchlists and custom fields |
| [Company Attachments](/api-reference/company-attachments) | Upload and manage files attached to companies |
| [Person Attachments](/api-reference/person-attachments) | Upload and manage files attached to people |
| [Company Global Fields](/api-reference/company-global-fields) | Manage custom fields across all companies |
| [Person Global Fields](/api-reference/person-global-fields) | Manage custom fields across all people |
| [Customer](/api-reference/customer) | Retrieve customer and user information |
