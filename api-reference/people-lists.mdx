title: People Lists
description: Manage people watchlists, custom fields, and list entries

People Lists (watchlists) let you organize and track collections of people with custom fields and views.

## Get people watchlists

<Info>**GET** `/watchlists/people`</Info>

Retrieve all people watchlists for your team.

```bash
curl 'https://api.harmonic.ai/watchlists/people' \
  -H 'apikey: your_api_key'
```

---

Retrieve a specific people watchlist.

### Path parameters

## Create people watchlist

<Info>**POST** `/watchlists/people`</Info>

Create a new people watchlist.

### Request body

<ParamField body="name" type="string" required>
  Name of the watchlist.
</ParamField>

<ParamField body="shared_with_team" type="boolean">
  Whether to share this list with your team.
</ParamField>

<ParamField body="people" type="array">
  Initial list of person IDs or URNs to add.
</ParamField>

### Example

```bash
curl -X POST 'https://api.harmonic.ai/watchlists/people' \
  -H 'apikey: your_api_key' \
  -H 'Content-Type: application/json' \
  -d '{"name": "Target Founders", "shared_with_team": true}'
```
