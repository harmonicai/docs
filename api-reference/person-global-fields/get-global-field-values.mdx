---
title: "Get global field values"
description: "Retrieve global field values for people"
---

<Info>This endpoint is available via **GraphQL only**. See the [GraphQL overview](/graphql/overview) for authentication and query details.</Info>

Retrieve the values of custom global fields for a specific person. Global fields are configured via the REST API and their values can be read through GraphQL.

```graphql
query GetPersonGlobalFieldValues {
  getPersonById(id: 67890) {
    name
    customFields {
      fieldUrn
      textValue
      numericValue
      dateValue
      singleSelectValue
      multiSelectValues
    }
  }
}
```
