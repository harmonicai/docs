<Info>This endpoint is available via **GraphQL only**. See the [GraphQL overview](/graphql/overview) for authentication and query details.</Info>

Retrieve all people connected to your team's network. Supports cursor-based pagination with a default of 10 results per page (maximum 100). You can filter by date range on connection activity.

```graphql
query GetPeopleInNetwork {
  getPeopleInNetwork(first: 10, after: null) {
    edges {
      node {
        name
        linkedinUrl
        currentTitle
        currentCompany {
          name
          domain
        }
      }
    }
    pageInfo {
      hasNextPage
      endCursor
    }
  }
}
```
