<Info>This endpoint is available via **GraphQL only**. See the [GraphQL overview](/graphql/overview) for authentication and query details.</Info>

Retrieve all companies connected to your team's network. Supports cursor-based pagination with a default of 10 results per page (maximum 100).

```graphql
query GetCompaniesInNetwork {
  getCompaniesInNetwork(first: 10, after: null) {
    edges {
      node {
        name
        domain
        connectionCount
      }
    }
    pageInfo {
      hasNextPage
      endCursor
    }
  }
}
```
