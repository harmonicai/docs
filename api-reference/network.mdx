The Network endpoints let you see how your team is connected to companies and people in the Harmonic database.

## Get team connections to a company

<Info>**GET** `/companies/{id_or_urn}/userConnections`</Info>

Retrieve your team's connections to a specific company.

### Path parameters

<ParamField path="id_or_urn" type="string" required>
  The company ID or URN.
</ParamField>

### Example

```bash
curl 'https://api.harmonic.ai/companies/12345/userConnections' \
  -H 'apikey: your_api_key'
```

## Get people in network

<Info>**GraphQL only**</Info>

Retrieve people in your team's network. Supports cursor-based pagination with a maximum of 100 results per page.

### Parameters

<ParamField query="first" type="integer" default="100">
  Number of results to return. Maximum: 100.
</ParamField>

<ParamField query="after" type="string">
  Pagination cursor.
</ParamField>

Additional filters are available for date range filtering on connection activity.

### Example query

```graphql
query {
  peopleInNetwork(first: 50) {
    edges {
      node {
        name
        linkedinUrl
        currentCompany {
          name
        }
      }
    }
    pageInfo {
      hasNext
      cursor
    }
  }
}
```

## Get companies in network

<Info>**GraphQL only**</Info>

Retrieve companies in your team's network. Supports cursor-based pagination with a maximum of 100 results per page.

### Parameters

<ParamField query="first" type="integer" default="100">
  Number of results to return. Maximum: 100.
</ParamField>

<ParamField query="after" type="string">
  Pagination cursor.
</ParamField>

### Example query

```graphql
query {
  companiesInNetwork(first: 50) {
    edges {
      node {
        name
        domain
        connectionCount
      }
    }
    pageInfo {
      hasNext
      cursor
    }
  }
}
```
