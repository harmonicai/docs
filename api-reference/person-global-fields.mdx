---
title: Person Global Fields
description: Manage custom fields that apply across all people
---

Person Global Fields let you define custom fields that are available across all person records, not just within a specific watchlist.

## Get person global fields

<Info>**GET** `/persons/custom_fields`</Info>

Retrieve all person global fields.

```bash
curl 'https://api.harmonic.ai/persons/custom_fields' \
  -H 'apikey: your_api_key'
```

---

## Create person global field

<Info>**POST** `/persons/custom_field`</Info>

Create a new person global field.

### Request body

<ParamField body="name" type="string" required>
  Name of the global field.
</ParamField>

<ParamField body="type" type="string" required>
  Field type. One of: `TEXT`, `NUMBER`, `DATE`, `SINGLE_SELECT`, `MULTI_SELECT`, `PERSON`, `WEBSITE`, `CHECKBOX`, `STATUS`.
</ParamField>

<ParamField body="metadata" type="object">
  Additional metadata.
</ParamField>

---

## Update person global field

<Info>**PUT** `/persons/custom_field`</Info>

Update an existing person global field.

### Request body

<ParamField body="custom_field_urn" type="string" required>
  The URN of the global field.
</ParamField>

<ParamField body="name" type="string">
  Updated name.
</ParamField>

<ParamField body="metadata" type="object">
  Updated metadata.
</ParamField>

---

## Delete person global field

<Info>**DELETE** `/persons/custom_field`</Info>

Delete a person global field and all associated values.

### Request body

<ParamField body="custom_field_urn" type="string" required>
  The URN of the global field to delete.
</ParamField>

---

## Upsert person global field values

<Info>**POST** `/persons/entries`</Info>

Set or update global field values for one or more people.

### Request body

<ParamField body="entries" type="array" required>
  Array of entries, each containing a person identifier and field values.
</ParamField>

---

## Get global field values

<Info>**GraphQL only**</Info>

Retrieve global field values for people. Available via GraphQL only.

```graphql
query {
  person(id: 67890) {
    name
    globalFields {
      fieldUrn
      value
    }
  }
}
```
