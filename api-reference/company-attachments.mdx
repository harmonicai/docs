---
title: Company Attachments
description: Upload, manage, and download files attached to companies
---

The Company Attachments endpoints let you manage file attachments on company records.

## Get attachments for companies

<Info>**POST** `/companies/attachments`</Info>

Retrieve attachments for one or more companies.

### Request body

<ParamField body="entity_urns" type="array[string]" required>
  Array of company URNs.
</ParamField>

### Example

```bash
curl -X POST 'https://api.harmonic.ai/companies/attachments' \
  -H 'apikey: your_api_key' \
  -H 'Content-Type: application/json' \
  -d '{"entity_urns": ["urn:harmonic:company:12345"]}'
```

---

## Initiate company attachment upload

<Info>**POST** `/companies/{id_or_urn}/attachments`</Info>

Start a file upload for a company. Returns a pre-signed upload URL.

### Path parameters

<ParamField path="id_or_urn" type="string" required>
  The company ID or URN.
</ParamField>

### Request body

<ParamField body="attachments" type="array" required>
  Array of attachment objects with `name`, `extension`, and `size` (in bytes).
</ParamField>

### Example

```bash
curl -X POST 'https://api.harmonic.ai/companies/12345/attachments' \
  -H 'apikey: your_api_key' \
  -H 'Content-Type: application/json' \
  -d '{"attachments": [{"name": "pitch-deck", "extension": "pdf", "size": 2048000}]}'
```

## Complete company attachment uploads

<Info>**PUT** `/companies/{id_or_urn}/attachments`</Info>
