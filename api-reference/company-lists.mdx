---
title: Company Lists
description: Manage company watchlists, custom fields, and list entries
---

Company Lists (watchlists) let you organize and track collections of companies with custom fields and views.

## Get watchlists

<Info>**GET** `/watchlists/companies`</Info>

Retrieve all company watchlists for your team.

```bash
curl 'https://api.harmonic.ai/watchlists/companies' \
  -H 'apikey: your_api_key'
```

---

## Get company watchlist

<Info>**GET** `/watchlists/companies/{id_or_urn}`</Info>

Retrieve a specific company watchlist.

### Path parameters

<ParamField path="id_or_urn" type="string" required>
  The watchlist ID or URN.
</ParamField>

```bash
curl 'https://api.harmonic.ai/watchlists/companies/12345' \
  -H 'apikey: your_api_key'
```

## Create company watchlist

<Info>**POST** `/watchlists/companies`</Info>

Create a new company watchlist.

### Request body

<ParamField body="name" type="string" required>
  Name of the watchlist.
</ParamField>

<ParamField body="shared_with_team" type="boolean">
  Whether to share this list with your team.
</ParamField>

<ParamField body="companies" type="array">
  Initial list of company IDs or URNs to add.
</ParamField>

### Example

```bash
curl -X POST 'https://api.harmonic.ai/watchlists/companies' \
  -H 'apikey: your_api_key' \
  -H 'Content-Type: application/json' \
  -d '{"name": "AI Startups", "shared_with_team": true}'
```
