---
title: Person Attachments
description: Upload, manage, and download files attached to people
---

The Person Attachments endpoints let you manage file attachments on person records.

## Get attachments for persons

<Info>**POST** `/persons/attachments`</Info>

Retrieve attachments for one or more persons.

### Request body

<ParamField body="entity_urns" type="array[string]" required>
  Array of person URNs.
</ParamField>

---

## Initiate person attachment upload

<Info>**POST** `/persons/{id_or_urn}/attachments`</Info>

Start a file upload for a person. Returns a pre-signed upload URL.

### Path parameters

<ParamField path="id_or_urn" type="string" required>
  The person ID or URN.
</ParamField>

### Request body

<ParamField body="attachments" type="array" required>
  Array of attachment objects with `name`, `extension`, and `size` (in bytes).
</ParamField>

---

## Complete person attachment uploads

<Info>**PUT** `/persons/{id_or_urn}/attachments`</Info>

Mark one or more attachment uploads as complete after uploading to the pre-signed URL.

---

## Rename person attachment

<Info>**PATCH** `/persons/{id_or_urn}/attachments/{urn}`</Info>

Rename an existing attachment.

### Path parameters

<ParamField path="id_or_urn" type="string" required>
  The person ID or URN.
</ParamField>

<ParamField path="urn" type="string" required>
  The attachment URN.
</ParamField>

---

## Get person attachment contents

<Info>**GET** `/persons/{id_or_urn}/attachments/{urn}`</Info>

Retrieve the download URL for an attachment.

---

## Delete person attachment

<Info>**DELETE** `/persons/{id_or_urn}/attachments/{urn}`</Info>

Permanently delete an attachment from a person.
