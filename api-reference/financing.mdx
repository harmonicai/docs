<Note>
Financing endpoints are available via **GraphQL only** and require a **paid add-on**. Contact [support@harmonic.ai](mailto:support@harmonic.ai) to enable access.
</Note>

The Financing endpoints provide access to investor profiles, their investments, and co-investor relationships.

## Get investor by URN

<Info>**GraphQL only**</Info>

Retrieve an investor profile by URN. Partners support cursor-based pagination. Investments and co-investors support offset-based pagination.

### Example query

```graphql
query {
  investor(urn: "urn:harmonic:investor:12345") {
    name
    description
    partners(first: 10) {
      edges {
        node {
          name
          linkedinUrl
        }
      }
      pageInfo {
        hasNext
        cursor
      }
    }
    investments(limit: 20, offset: 0) {
      company {
        name
      }
      roundDate
      roundType
    }
  }
}
```

## Get investors by URNs

<Info>**GraphQL only**</Info>

Retrieve multiple investor profiles in a single query.

### Example query

```graphql
query {
  investors(urns: ["urn:harmonic:investor:123", "urn:harmonic:investor:456"]) {
    name
    description
  }
}
```

## Get investor by canonical URL

<Info>**GraphQL only**</Info>

Look up an investor by their canonical URL (e.g., their website or Crunchbase profile).

### Parameters

<ParamField query="url" type="string" required>
  The canonical URL of the investor.
</ParamField>

### Example query

```graphql
query {
  investorByCanonical(url: "https://example-vc.com") {
    name
    urn
    description
  }
}
```
